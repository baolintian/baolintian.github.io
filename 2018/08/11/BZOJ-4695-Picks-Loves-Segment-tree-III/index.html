BZOJ 4695 Picks Loves Segment tree III

BZOJ 4695 Picks Loves Segment tree III

tag: 数据结构 线段树 区间更新 区间修改

题目分析

我们要完成以下数据结构：

  • 1.给一个区间 $[L,R]$ 加上一个数 $x$

  • 2.把一个区间 $[L,R]$ 里的数变成 $\max (x, A_i)$

  • 3.把一个区间 $[L,R]$ 里的数变成 $\min(x, A_i)$

  • 4.求区间 $[L,R]$ 的和

  • 5.求区间 $[L,R]$ 的最大值

  • 6.求区间 $[L,R]$ 的最小值

这道题我们可以参考吉老师的论文，他证明以上 $6$ 种操作可以在 $O(m \log ^ 2 n)$ 的复杂度下进行。
